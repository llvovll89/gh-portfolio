## 1. Figma란?

Figma는 **브라우저 기반의 UI/UX 디자인 협업 툴**입니다.
별도 설치 없이 웹에서 실행되며, 실시간 멀티 유저 편집이 가능합니다.

| 특징 | 설명 |
|------|------|
| 플랫폼 | 웹 브라우저 / 데스크탑 앱 |
| 협업 | 실시간 동시 편집 |
| 가격 | 무료 플랜 제공 (팀 기능은 유료) |
| 포맷 | .fig (자체 포맷), SVG, PNG 등 내보내기 가능 |

## 2. 기본 인터페이스

```
┌─────────────────────────────────────────────┐
│  툴바 (상단)                                 │
├──────────┬──────────────────────┬────────────┤
│ 레이어   │                      │ 속성 패널  │
│ 패널     │    캔버스 영역        │ (우측)     │
│ (좌측)   │                      │            │
└──────────┴──────────────────────┴────────────┘
```

- **툴바 (상단)**: 이동, 도형, 펜, 텍스트, 핸드 도구 등
- **레이어 패널 (좌측)**: 페이지, 레이어, 컴포넌트 목록
- **캔버스**: 실제 디자인 작업 영역
- **속성 패널 (우측)**: 선택된 요소의 크기, 색상, 정렬 등

## 3. 기본 도형 & 텍스트 다루기

### 도형 그리기

| 단축키 | 도구 |
|--------|------|
| `R` | 사각형(Rectangle) |
| `O` | 원(Ellipse) |
| `L` | 선(Line) |
| `P` | 펜(Pen) |
| `T` | 텍스트 |

### 색상 및 스타일

- 우측 패널 → **Fill**에서 색상 설정
- HEX, RGB, HSL 모두 입력 가능
- **Stroke** 옵션으로 테두리 추가
- **Opacity** 슬라이더로 투명도 조정

### 텍스트 설정

```
폰트 패밀리 → 폰트 사이즈 → 굵기
줄 간격(Line Height) → 자간(Letter Spacing)
정렬: 좌/중앙/우
```

## 4. 프레임(Frame)과 그룹(Group)

### 프레임(Frame)
- 단축키: `F`
- 디바이스 프리셋 선택 가능 (iPhone 16, Desktop 등)
- **클립 콘텐츠**: 프레임 밖으로 나간 요소를 숨김
- 컴포넌트, 오토 레이아웃의 기반이 됨

### 그룹(Group)
- 단축키: `Ctrl + G`
- 여러 요소를 묶어 일괄 이동/조절
- 프레임과 달리 독립적인 크기 기준이 없음

> **TIP**: UI 디자인에는 그룹보다 **프레임 사용을 권장**합니다.

## 5. 컴포넌트(Component)

반복 사용되는 UI 요소를 컴포넌트로 만들면 수정 시 일괄 적용됩니다.

### 컴포넌트 만들기

```
요소 선택 → 우클릭 → Create Component
또는 Ctrl + Alt + K
```

### 마스터 컴포넌트 vs 인스턴스

| 구분 | 설명 |
|------|------|
| 마스터 컴포넌트 | 원본. 수정 시 모든 인스턴스에 반영 |
| 인스턴스 | 복사본. 개별 속성(텍스트, 색상) 오버라이드 가능 |

### Variants (변형)

하나의 컴포넌트에 여러 상태(Default, Hover, Active, Disabled)를 묶어 관리.

```
컴포넌트 선택 → 우측 패널 → Add Variant
```

## 6. 오토 레이아웃(Auto Layout)

요소들을 자동으로 정렬하고 간격을 유지하는 기능입니다.

### 적용 방법

```
요소 선택 → Shift + A
또는 우측 패널 → Auto Layout 추가
```

### 주요 설정

| 설정 | 설명 |
|------|------|
| Direction | 가로(Horizontal) / 세로(Vertical) |
| Spacing | 요소 간 간격 |
| Padding | 내부 여백 (상하좌우) |
| Wrap | 줄바꿈 여부 |
| Alignment | 정렬 방식 |

### Hug vs Fill vs Fixed

- **Hug**: 내용 크기에 맞게 자동 조절
- **Fill**: 부모 컨테이너를 꽉 채움
- **Fixed**: 고정 크기

## 7. 프로토타이핑

화면 간 인터랙션을 연결해 실제 앱처럼 테스트할 수 있습니다.

### 연결 방법

```
1. Prototype 탭 클릭 (우측 패널 상단)
2. 요소의 우측 파란 점을 드래그
3. 연결할 대상 프레임에 드롭
```

### 트리거 종류

| 트리거 | 설명 |
|--------|------|
| On Click | 클릭 시 |
| On Hover | 호버 시 |
| On Drag | 드래그 시 |
| After Delay | 일정 시간 후 |

### 트랜지션 효과

- Instant, Dissolve, Smart Animate, Slide In/Out, Push 등

### 프리뷰 실행

```
우측 상단 ▶ (Present) 버튼 클릭
```

## 8. 협업 기능

### 공유 및 권한

```
우측 상단 Share → 링크 복사 또는 이메일 초대
권한: View only / Can Edit
```

### 실시간 커서

- 같은 파일에 있는 팀원의 커서가 실시간으로 표시됨
- 팀원 아바타 클릭 시 해당 위치로 이동

### 댓글(Comment)

```
단축키: C → 캔버스 클릭 → 댓글 작성
특정 요소에 피드백 남기기 가능
```

### 개발자 핸드오프

```
우측 상단 → Dev Mode 활성화
→ 개발자가 CSS, iOS, Android 코드 스펙 확인 가능
```

## 9. 플러그인 추천

| 플러그인 | 용도 |
|----------|------|
| **Unsplash** | 무료 이미지 삽입 |
| **Iconify** | 아이콘 라이브러리 |

| **Figma to HTML** | HTML/CSS 변환 |
| **Color Palettes** | 색상 팔레트 생성 |
| **Stark** | 접근성(대비율) 검사 |

### 플러그인 설치

```
메뉴(햄버거) → Plugins → Browse plugins in Community
```

## 10. 단축키 정리

### 도구 선택

| 단축키 | 기능 |
|--------|------|
| `V` | 선택 도구 |
| `H` | 핸드 도구 (캔버스 이동) |
| `R` | 사각형 |
| `O` | 원 |
| `T` | 텍스트 |
| `F` | 프레임 |
| `P` | 펜 |

### 편집

| 단축키 | 기능 |
|--------|------|
| `Ctrl + C / V` | 복사 / 붙여넣기 |
| `Ctrl + D` | 제자리 복제 |
| `Ctrl + G` | 그룹 |
| `Ctrl + Alt + K` | 컴포넌트 만들기 |
| `Ctrl + Z` | 실행 취소 |
| `Ctrl + Shift + H` | 수평 가운데 정렬 |
| `Ctrl + Shift + V` | 수직 가운데 정렬 |

### 뷰 조작

| 단축키 | 기능 |
|--------|------|
| `Ctrl + 0` | 100% 줌 |
| `Ctrl + Shift + 0` | 선택 요소에 맞게 줌 |
| `Space + 드래그` | 캔버스 패닝 |
| `Ctrl + \` | UI 숨기기/보이기 |

## 마무리

Figma는 현재 UI/UX 디자인 업계의 표준 툴로 자리잡았습니다.
개발자도 기본 사용법을 익혀두면 디자이너와 협업 시 큰 도움이 됩니다.

> **핵심 순서**: 프레임 → 컴포넌트 → 오토 레이아웃 → 프로토타입 순으로 익히면 효율적입니다.
