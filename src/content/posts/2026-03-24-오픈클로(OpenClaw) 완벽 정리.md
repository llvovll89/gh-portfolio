## 1. 오픈클로란 무엇인가?

**오픈클로(OpenClaw)**는 로컬 컴퓨터에서 구동되는 자율형 오픈소스 AI 에이전트입니다.

단순히 질문에 답하는 AI 챗봇이 아니라, **직접 행동(Action)**을 취하는 에이전트입니다. 이메일을 보내고, 파일을 정리하고, 코드를 작성하고 실행하며, 웹을 검색하는 등 컴퓨터로 할 수 있는 거의 모든 작업을 대신 처리합니다.

- **라이선스:** MIT (완전 무료)
- **공식 GitHub:** [github.com/openclaw/openclaw](https://github.com/openclaw/openclaw)
- **GitHub 스타:** 310,000+ ⭐ (2026년 3월 기준)

## 2. 탄생 배경과 역사

| 시기 | 사건 |
|------|------|
| 2025년 11월 | 오스트리아 개발자 **Peter Steinberger**가 'WhatsApp Relay' 프로젝트로 시작 |
| 2025년 11월 | 'Clawdbot'이라는 이름으로 GitHub 공개, 일주일 만에 ⭐ 10만 달성 |
| 2025년 12월 | Anthropic 상표 문제로 'Moltbot'으로 이름 변경 |
| 2026년 초 | 다시 'OpenClaw'로 최종 변경 |
| 2026년 2월 14일 | 창시자 Peter Steinberger가 OpenAI에 합류, 프로젝트는 독립 오픈소스 재단으로 이관 |
| 2026년 3월 | 젠슨 황이 GTC 2026 기조연설에서 "인류 역사상 가장 성공적인 오픈소스 프로젝트"로 언급 |

Peter Steinberger는 **PSPDFKit(현 Nutrient)**의 창립자로 잘 알려진 iOS/macOS 개발자였습니다. 개인 프로젝트로 시작한 것이 전 세계적인 현상이 된 셈입니다.

## 3. 기존 AI 챗봇과의 차이점

| 구분 | 기존 AI 챗봇 (ChatGPT, Claude 등) | 오픈클로 (OpenClaw) |
|------|----------------------------------|---------------------|
| **동작 방식** | 질문 → 텍스트 응답 | 지시 → 실제 작업 수행 |
| **파일 접근** | 불가 (업로드 제한적) | 로컬 파일 시스템 전체 접근 |
| **실행 환경** | 클라우드 서버 | 로컬 컴퓨터 |
| **비용** | 유료 구독 필요 | 무료 (오픈소스) |
| **개인정보** | 서버에 데이터 전송 | 로컬에서만 처리 |
| **메신저 연동** | 제한적 | 30개 이상 플랫폼 지원 |

## 4. 주요 기능

### 자율 작업 처리

```
사용자: "지난주 이메일 중 미응답 건 정리해서 답장 초안 만들어줘"
오픈클로: → 이메일함 접근 → 필터링 → 초안 작성 → 확인 요청
```

### 메모리 기능

- 사용자 선호사항, 작업 패턴, 개인 정보를 **영구적으로 기억**
- Markdown 파일에 자동 기록
- 매번 설명할 필요 없이 맥락 유지

### 멀티 에이전트

- 여러 에이전트가 협력해 복잡한 작업을 분업 처리
- **OpenClaw Office** 같은 시각화 도구로 에이전트 협업 모니터링 가능

### 메신저 인터페이스

- WhatsApp, Telegram, Discord, Slack 등으로 **문자를 보내듯** AI에게 지시
- 에디터를 켤 필요 없이 스마트폰에서 바로 컴퓨터 제어

## 6. 설치 방법

### 사전 요구사항

- **Node.js** (v18 이상 권장)
- macOS, Windows, Linux 모두 지원 (macOS 가장 설치 간편)

### macOS 설치

```bash
# 1. Homebrew 설치 (미설치 시)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 2. Node.js 설치
brew install node

# 3. 오픈클로 설치
npm i -g openclaw

# 4. 초기 설정
openclaw onboard
```

### Windows 설치

```bash
# 1. Node.js 공식 사이트에서 설치 (nodejs.org)

# 2. 관리자 권한으로 PowerShell 실행 후
npm i -g openclaw

# 3. 초기 설정
openclaw onboard
```

> **팁:** Windows에서는 추가 환경 변수 설정이 필요할 수 있습니다. `openclaw doctor` 명령어로 문제를 자동 진단하세요.

## 7. 기본 명령어

| 명령어 | 설명 |
|--------|------|
| `openclaw onboard` | 최초 설정 마법사 실행 |
| `openclaw gateway` | AI 에이전트 실행 (메신저 연결 시작) |
| `openclaw doctor` | 설정 오류 자동 진단 및 수정 |
| `openclaw update` | 최신 버전으로 업데이트 |
| `openclaw stop` | 에이전트 중지 |

## 9. 보안 주의사항

오픈클로는 강력한 만큼 주의도 필요합니다.

### 주요 위험

- **파일 유출:** 악의적인 프롬프트로 개인 파일이 외부에 전송될 수 있음
- **암호화폐 지갑:** 지갑 파일 접근 후 유출 사례 보고
- **셸 명령 실행:** 잘못된 지시 시 시스템 손상 가능

### 권장 사항

```
✅ 신뢰할 수 없는 사람에게 메신저 접근 권한 부여 금지
✅ 민감한 파일은 오픈클로 접근 경로 외부에 보관
✅ 암호화폐 지갑, SSH 키 등은 별도 보안 폴더 격리
✅ 회사 업무용 PC에는 설치 비권장
```

> **주의:** 네이버, 카카오 등 주요 IT 기업에서는 사내 PC 설치를 **전면 금지**하고 있습니다.
