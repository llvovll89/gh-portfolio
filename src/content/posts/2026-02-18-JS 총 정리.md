## 1. 변수와 스코프

### 1-1. var, let, const

- `var`: 함수 스코프, 호이스팅(초기화 undefined)
- `let`/`const`: 블록 스코프, 재선언 불가, `const`는 재할당 불가

#### 실습 예제

```js
function testScope() {
  if (true) {
    var a = 1;
    let b = 2;
    const c = 3;
  }
  console.log(a); // 1
  // console.log(b); // ReferenceError
  // console.log(c); // ReferenceError
}
testScope();
```

#### 문제

아래 코드의 출력 결과를 예측해보세요.

```js
console.log(x);
var x = 5;
let y = 10;
console.log(y);
```

<details>
<summary>정답/해설</summary>

- 첫 번째 `console.log(x)`는 `undefined` (var 호이스팅)
- `let y = 10;` 이후 `console.log(y)`는 `10`

</details>

---

### 1-2. 스코프와 호이스팅

- 함수 스코프 vs 블록 스코프
- 호이스팅: 선언이 코드 상단으로 끌어올려짐

#### 실습 예제

```js
function hoist() {
  console.log(a); // undefined
  var a = 1;
}
hoist();
```

## 2. 자료형과 연산자

### 2-1. 원시/참조 타입

- 원시: string, number, boolean, null, undefined, symbol, bigint
- 참조: object, array, function 등

#### 실습 예제

```js
let a = 1;
let b = a;
b = 2;
console.log(a); // 1

let obj1 = { x: 1 };
let obj2 = obj1;
obj2.x = 2;
console.log(obj1.x); // 2
```

---

### 2-2. typeof, == vs ===

- `typeof null === 'object'` (역사적 버그)
- `==`(느슨), `===`(엄격) 비교

#### 문제

아래 코드의 결과는?

```js
console.log(null == undefined); // ?
console.log(null === undefined); // ?
```

<details>
<summary>정답/해설</summary>

- `true`, `false`

</details>

---

### 2-3. 최신 연산자

- 옵셔널 체이닝: `obj?.prop`
- 널 병합: `a ?? b`
- 논리 할당: `a ||= b`, `a &&= b`, `a ??= b`

#### 실습 예제

```js
const user = { profile: { name: "Lee" } };
console.log(user?.profile?.name); // "Lee"
console.log(user?.address?.city); // undefined

let val;
val ??= "default";
console.log(val); // "default"
```

## 3. 함수

### 3-1. 함수 선언식/표현식

```js
function foo() { return 1; }
const bar = function() { return 2; };
```

### 3-2. 화살표 함수

- `this` 바인딩 없음

#### 실습 예제

```js
const obj = {
  value: 10,
  getValue: () => this.value,
  getValueFn() { return this.value; }
};
console.log(obj.getValue()); // undefined
console.log(obj.getValueFn()); // 10
```

---

### 3-3. 기본 매개변수, rest

```js
function greet(name = "익명") {
  return `안녕, ${name}`;
}
console.log(greet()); // "안녕, 익명"

function sum(...nums) {
  return nums.reduce((a, b) => a + b, 0);
}
console.log(sum(1,2,3)); // 6
```

## 5. 클래스와 상속

### 5-1. 클래스 선언, 필드

```js
class Animal {
  #privateField = 0;
  constructor(name) {
    this.name = name;
  }
  speak() {
    return `${this.name}이(가) 소리를 냅니다.`;
  }
}
```

### 5-2. 상속

```js
class Dog extends Animal {
  speak() {
    return `${this.name}가 멍멍!`;
  }
}
```

#### 실습 예제

```js
const d = new Dog("초코");
console.log(d.speak()); // "초코가 멍멍!"
```

## 7. 비동기 처리

### 7-1. 콜백, Promise

```js
function wait(ms) {
  return new Promise(res => setTimeout(res, ms));
}
wait(1000).then(() => console.log("1초 후"));
```

### 7-2. async/await, fetch

```js
async function getData() {
  const res = await fetch('/api/data');
  return res.json();
}
```

#### 문제

아래 코드의 결과는?

```js
async function f() {
  return 1;
}
f().then(console.log);
```

<details>
<summary>정답/해설</summary>

- `1` (async 함수는 항상 Promise 반환)

</details>

## 9. 이터러블/이터레이터, 제너레이터

### 9-1. 이터러블/이터레이터

```js
const arr = [1,2,3];
for (const v of arr) console.log(v);
```

### 9-2. 제너레이터

```js
function* gen() {
  yield 1;
  yield 2;
}
const g = gen();
console.log(g.next()); // { value:1, done:false }
```

## 10. Proxy / Reflect

```js
const target = {};
const proxy = new Proxy(target, {
  get(obj, prop) {
    console.log('get', prop);
    return Reflect.get(obj, prop);
  }
});
proxy.foo = 1;
console.log(proxy.foo); // get foo, 1
```
