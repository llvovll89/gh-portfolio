# 컴퓨터 하드웨어 정리 (실전 조립 기준)

PC를 조립하거나 업그레이드할 때 알아야 할 **핵심 하드웨어 구성요소**를 "개념 → 스펙 읽는 법 → 선택 팁" 흐름으로 정리한 노트입니다.

> 목표: 하드웨어 스펙 보고 **막히지 않고 이해하기**

---

## 목차

1) CPU (중앙처리장치)
2) GPU (그래픽카드)
3) 메인보드 (마더보드)
4) RAM (메모리)
5) 저장장치 (SSD/HDD)
6) 파워서플라이 (PSU)
7) 케이스 & 쿨러
8) 조립 체크리스트

---

## 1) CPU (Central Processing Unit)

### 1.1 핵심 개념

**CPU는 컴퓨터의 두뇌**로, 모든 연산과 명령 처리를 담당합니다.

주요 제조사:
- **Intel**: Core i3/i5/i7/i9 시리즈 (13세대, 14세대 등)
- **AMD**: Ryzen 3/5/7/9 시리즈 (7000, 9000 시리즈 등)

### 1.2 스펙 읽는 법

```
Intel Core i7-13700K
         ↑   ↑  ↑  ↑
      시리즈 등급 세대 모델
```

- **코어 수 (Cores)**: 물리적 연산 유닛 (예: 8코어, 16코어)
- **쓰레드 수 (Threads)**: 동시 처리 가능 작업 수 (하이퍼스레딩)
- **클럭 속도 (GHz)**: 처리 속도 (기본/부스트)
- **TDP (W)**: 소비 전력 및 발열

### 1.3 선택 팁

| 용도 | 추천 |
|---|---|
| 문서/웹서핑 | i3/Ryzen 3 |
| 개발/멀티태스킹 | i5/Ryzen 5 |
| 게임/영상편집 | i7/Ryzen 7 |
| 전문작업/렌더링 | i9/Ryzen 9 |

> **주의**: CPU 소켓이 메인보드와 호환되는지 반드시 확인 (Intel LGA1700, AMD AM5 등)

---

## 2) GPU (Graphics Processing Unit)

### 2.1 핵심 개념

**그래픽 처리 전문 칩**으로, 게임/영상/AI 학습에 필수입니다.

주요 제조사:
- **NVIDIA**: GeForce RTX 시리즈 (4090, 4080, 4070 등)
- **AMD**: Radeon RX 시리즈 (7900 XTX, 7800 XT 등)

### 2.2 스펙 읽는 법

```
NVIDIA GeForce RTX 4070 Ti 12GB
               ↑   ↑    ↑   ↑
             시리즈 모델 변형 VRAM
```

- **VRAM (GB)**: 그래픽 메모리 (8GB, 12GB, 24GB 등)
- **CUDA 코어 / Stream Processor**: 병렬 연산 유닛 수
- **TDP (W)**: 소비 전력 (300W 이상도 흔함)

### 2.3 선택 팁

| 용도 | 추천 |
|---|---|
| 내장 그래픽으로 충분 | CPU 내장 (Intel UHD/AMD Radeon) |
| 1080p 게임 | RTX 4060 / RX 7600 |
| 1440p 게임 | RTX 4070 / RX 7800 XT |
| 4K 게임/AI/영상 | RTX 4080/4090 / RX 7900 XTX |

> **팁**: 모니터 해상도와 주사율(Hz)을 먼저 정하고 GPU 선택

---

## 3) 메인보드 (Motherboard)

### 3.1 핵심 개념

**모든 부품을 연결하는 기판**입니다.

주요 칩셋:
- Intel: Z790, B760 등
- AMD: X670E, B650 등

### 3.2 주요 스펙

- **소켓**: CPU와 호환 여부 (LGA1700, AM5 등)
- **폼팩터**: 크기 규격
  - ATX: 표준 크기
  - Micro-ATX: 작은 크기
  - Mini-ITX: 초소형
- **RAM 슬롯**: DDR4/DDR5 지원 여부, 최대 용량
- **확장 슬롯**: PCIe 4.0/5.0, M.2 NVMe 슬롯 개수
- **USB/네트워크**: USB 3.2, Wi-Fi 6E, 2.5G LAN 등

### 3.3 선택 팁

- **오버클럭 필요**: Z/X 칩셋 (Intel Z790, AMD X670)
- **가성비**: B 칩셋 (Intel B760, AMD B650)
- **형태**: 케이스 크기에 맞춰 선택

---

## 4) RAM (Random Access Memory)

### 4.1 핵심 개념

**임시 작업 공간**으로, 많을수록 멀티태스킹에 유리합니다.

현재 주류:
- **DDR4**: 3200MHz ~ 3600MHz
- **DDR5**: 5200MHz ~ 6400MHz (최신 플랫폼)

### 4.2 스펙 읽는 법

```
DDR5-6000 32GB (16GB x 2) CL30
 ↑     ↑    ↑      ↑        ↑
규격  속도  용량   구성   레이턴시
```

- **용량**: 16GB, 32GB, 64GB 등
- **속도 (MHz)**: 클럭 속도
- **CL (CAS Latency)**: 낮을수록 빠름

### 4.3 선택 팁

| 용도 | 추천 |
|---|---|
| 문서/웹서핑 | 16GB |
| 개발/게임 | 32GB |
| 영상편집/VM | 64GB+ |

> **주의**: 메인보드가 지원하는 DDR4/DDR5 규격 확인 필수
> **듀얼채널**: 같은 용량 2개 구성 (16GBx2) 권장

---

## 5) 저장장치 (SSD / HDD)

### 5.1 핵심 개념

- **SSD (Solid State Drive)**: 빠름, 비쌈
- **HDD (Hard Disk Drive)**: 느림, 저렴, 대용량 보관용

### 5.2 SSD 종류

**인터페이스별 분류**:
- **SATA SSD**: 최대 550MB/s (구형)
- **NVMe SSD (M.2)**: 3,000~7,000MB/s (주류)
  - PCIe 3.0: ~3,500MB/s
  - PCIe 4.0: ~7,000MB/s
  - PCIe 5.0: ~14,000MB/s (최신)

### 5.3 용량 권장

```
OS + 프로그램: 500GB NVMe (최소)
게임/라이브러리: 1TB ~ 2TB NVMe
대용량 보관: 2TB ~ 4TB HDD
```

### 5.4 선택 팁

- **OS 설치용**: NVMe SSD 필수
- **게임/개발**: 1TB NVMe 추천
- **백업/미디어**: HDD로 충분

> **체감 차이**: SATA → NVMe 업그레이드 시 부팅/로딩 속도 2~3배 향상

---

## 6) 파워서플라이 (PSU)

### 6.1 핵심 개념

**모든 부품에 전원 공급**하는 장치입니다.

주요 인증:
- **80 PLUS Bronze/Silver/Gold/Platinum**: 효율 등급
- **모듈러 방식**: 케이블 탈부착 가능 여부

### 6.2 용량 계산

```
CPU (150W) + GPU (300W) + 기타 (150W) = 600W
→ 여유 20% → 720W 이상 권장
```

실전 권장:
- **저전력 (내장그래픽)**: 450W
- **미드레인지 (RTX 4060 급)**: 650W
- **하이엔드 (RTX 4080 급)**: 850W
- **최상급 (RTX 4090 급)**: 1000W+

### 6.3 선택 팁

- **인증**: 최소 80 PLUS Bronze 이상
- **용량**: 실제 소비 + 20% 여유
- **제조사**: Corsair, Seasonic, EVGA 등 신뢰도 확인

> **주의**: 저품질 PSU는 시스템 전체를 위험에 빠뜨릴 수 있음

---

## 7) 케이스 & 쿨러

### 7.1 케이스

**폼팩터** 확인:
- ATX 케이스 → ATX/Micro-ATX/Mini-ITX 보드 호환
- Micro-ATX 케이스 → Micro-ATX/Mini-ITX만 가능

**체크 포인트**:
- GPU 최대 길이 (320mm, 380mm 등)
- CPU 쿨러 최대 높이
- 전면/후면 팬 수
- USB/오디오 포트 위치

### 7.2 CPU 쿨러

**종류**:
- **공랭 쿨러**: 저렴, 관리 쉬움
- **수랭 쿨러**: 조용함, 냉각 우수, 비쌈

**TDP 확인**:
- CPU TDP ≤ 쿨러 TDP 커버 범위

예시:
```
CPU: i7-13700K (TDP 125W)
→ 쿨러: TDP 150W 이상 권장
```

## 8) 조립 체크리스트

### 8.1 호환성 확인

- [ ] CPU 소켓 = 메인보드 소켓
- [ ] RAM 규격 (DDR4/DDR5) = 메인보드 지원
- [ ] GPU 길이 < 케이스 최대 길이
- [ ] PSU 용량 >= (CPU + GPU 소비 전력) x 1.2
- [ ] M.2 슬롯 개수 확인

### 8.2 조립 순서 (권장)

1. **메인보드에 CPU 장착** + 쿨러 장착
2. **RAM 삽입** (듀얼채널: 2/4번 슬롯 우선)
3. **M.2 SSD 장착** (방열판 확인)
4. **메인보드를 케이스에 고정**
5. **PSU 장착** 및 케이블 연결
6. **GPU 장착** (PCIe 슬롯)
7. **케이블 정리** 및 팬 연결
8. **전원 투입 전 재확인**

### 8.3 첫 부팅 체크

```
1. 모니터 신호 확인 (HDMI/DP)
2. BIOS 진입 (DEL/F2 키)
3. CPU/RAM/저장장치 인식 확인
4. BOOT 순서 설정
5. OS 설치
```

## 9) 추가 팁

### 9.1 부품 구매 순서

1. **CPU** 먼저 선택 (인텔/AMD 결정)
2. **메인보드** (소켓 호환)
3. **RAM** (DDR4/5 확인)
4. **GPU** (용도와 예산)
5. **저장장치** (NVMe 우선)
6. **PSU** (용량 계산)
7. **케이스/쿨러** (크기 확인)

### 9.2 벤치마크 사이트

- **CPU**: PassMark, Cinebench
- **GPU**: 3DMark, UserBenchmark
- **통합**: PC Part Picker (호환성 체크)

### 9.3 실수 방지

- RAM 미장착 (클릭음 확인)
- 24핀/8핀 전원 미연결
- CPU 쿨러 보호필름 제거 안함
- GPU 보조전원 미연결

## 정리

| 부품 | 첫 업그레이드 우선순위 |
|---|---|
| **CPU** | 세대 점프 시 체감 큼 |
| **GPU** | 게임/영상 작업 시 최우선 |
| **RAM** | 16GB → 32GB 멀티태스킹 체감 |
| **SSD** | SATA → NVMe 체감 명확 |
| **PSU** | 용량 부족 시만 교체 |

> 하드웨어는 **용도 우선**으로 선택하고, 과도한 스펙보다 **균형 잡힌 구성**이 중요합니다.
> 조립 전 반드시 **호환성 체크** + **정전기 방지** 필수!
